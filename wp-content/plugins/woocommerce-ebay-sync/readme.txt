== Description ==

WooCommerce to eBay integration.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ebaylink` directory, install via. Upload Plugin, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the eBay Sync Settings screen to configure the plugin.

== Frequently Asked Questions ==

= How do I sync my products to eBay? =

After completing the Settings pages and mapping your products/categories to eBay categories, the products will by synced via. Wordpress' cron functionality.
